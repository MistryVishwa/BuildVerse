# Student Health Assistant

A centralized dashboard helping students monitor hydration, sleep, exercise, screen time, and habits for a balanced lifestyle. Built with HTML, CSS, and Vanilla JS, using Local Storage for data persistence.

## Features
- **Health Dashboard**: Overview of wellness scores and daily metrics.
- **Trackers**: Modules for Hydration, Sleep, Exercise, and Screen Time.
- **Habit Manager**: Add custom habits and track your streaks.
- **Dark/Light Mode**: Fully responsive modern UI that respects your eyes.
