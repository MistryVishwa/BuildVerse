🛡️ Security Best Practices
🎲
Use a passphrase
A random string of 4+ words is easy to remember and extremely hard to crack. Example: maple-orbit-piano-7
🔑
Unique per site
Never reuse passwords. A breach on one site will compromise all others if you do. Use a password manager.
🔄
Enable 2FA
Two-factor authentication adds a second layer of defense even if your password is leaked.
📋
Use a password manager
Tools like Bitwarden, 1Password, or KeePass generate and store strong, unique passwords securely.
🚫
Avoid personal info
Birthdays, names, and pet names are the first things attackers try in targeted attacks.
⏰
Change if breached
Check haveibeenpwned.com regularly. If your email appears in a breach, change affected passwords immediately.
