# Grid Recall Challenge

A premium, browser-based memory puzzle game built with vanilla HTML, CSS, and JavaScript.

## How to Play

1. **Memorize** – Watch the glowing purple cells during the countdown.
2. **Recall** – Click every cell you remember in any order.
3. **Submit** – Hit the Submit button to check your answer.
4. **Repeat** – Nail it perfectly to advance to the next level!

## Features

- 🧠 **Progressive difficulty** – Grid grows from 3×3 up to 7×7 over time
- 🔥 **Streak system** – Perfect recall chains earn bonus points
- 🌟 **Streak rewards** – Hit 3, 5, 7, 10, or 15 streaks for milestone bonuses
- 🎊 **Confetti celebrations** – Burst of confetti on milestone streaks
- ⏱ **Countdown timer** – Animated timer shows memorize time remaining
- 💾 **High-score persistence** – Best score saved in localStorage
- 🔗 **Share your score** – Native share / clipboard fallback
- 🌌 **Animated starfield** – Twinkling background stars
- ♿ **Accessible** – ARIA roles, labels, and keyboard-friendly UI

## Tech Stack

- Pure HTML5 / CSS3 / Vanilla JavaScript
- No dependencies or build steps
- Google Fonts (Outfit)

## Run Locally

Just open `index.html` in any modern browser — no server needed!
