# Focus Timer (Pomodoro)

A mechanical, brass-instrument-styled Pomodoro timer with a draggable dial for custom durations, a task list linked to sessions, a session ledger, and a stats dashboard for staying focused.

## Features
- **Timer Dial**: 25/5, 50/10, and 90/15 presets, plus a draggable brass knob for any custom duration.
- **Task List**: Add tasks, attach them to a session, and track completed-pomodoro counts.
- **Session Ledger**: Every completed focus/break session logged with time and duration.
- **Statistics Dashboard**: Daily focus minutes, sessions completed, streaks, and a 7-day chart.
- **Sound Notifications**: A bell plays on session completion, toggleable on/off.

## Built with

HTML, CSS, and vanilla JS (single file). Data is stored in `localStorage`, so it persists between visits on the same browser.
